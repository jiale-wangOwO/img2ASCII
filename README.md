# Image to ASCII Art

A Python project to convert images into ASCII art.

## Requirements

- Python 3.x
- Pillow
- NumPy

## How to use

1. Put your image in the `assets/` folder.
2. Run the script `src/ascii_art.py` to generate the ASCII art.
3. The output will be saved in the `output/` folder.
